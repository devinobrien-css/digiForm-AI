# digiForm-AI
A forum extraction &amp; document organization SAAS application

## Project Dependencies

This project requires:

```
React 18.2
TailwindCSS
AWS CLI(for profiles)
AWS SAM CLI
Nodejs 18.x
Python3.9
```
